# Agrohealth
Agrohealth is web-app that focuses on agriculture and health sector. It fulfils the basic need of a farmer via providing them better assessment for better crop production and a good harvest at the end. In the health sector it provides stroke prediction and life expectancy prediction. We have used machine learning technology and flask for interface.
